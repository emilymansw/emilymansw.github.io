Project Overview

Curated directory of the best notion tools and templates. Built with Next.js, KeystoneJS and PostgreSQL. With headless CMS & GraphQL API powered by KeystoneJS.

Project Highlight


Infinite Scroll:
Using Intersection Observer API and useRef hook to observe the element position, when the observer detects the element reaches the bottom, calls fetchMore from the Apollo Client useQuery hook to add more 9 items to the existing list until all items are fetched.



Filter:
The implementation of the filter function is connecting the state of the checkboxes to Next.js router query. And parsing the query object value to update the GraphQL variables.


Masonry Layout:
Using the CSS column-count Property to achieve Pinterest style image gallery to showcase the templates collection.
