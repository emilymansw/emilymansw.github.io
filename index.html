About Me

Get to know me!

Hi! I am an experienced marketer decided to make a career change that is more in line with my passions. In February, I quitted my job to learn programming.

Over the past months, I completed the Java Developer and Java Backend Developer programs on JetBrains Academy. I acquired knowledge in data structures, OOP concepts, Spring Boot framework, REST API, SQL and database design.

With the learn by doing approach, for my coursework, I created several simple console-based applications including Cinema room manager, Budget manager, Bulls and Cows, Tic-Tac-Toe with AI, also some simple backend servers including an account service app and recipes app.

I continued my learning in web development, and dived into frontend development, learning Javascript, Typescript, React and other technologies. To showcase my skillset, I created two full-stack applications, the Skateshop e-commerce app and Notionhut, a directory app.
