Project Overview

Full Stack E-Commerce Application with Admin Panel built with React, Spring Boot. Integrated with Stripe checkout, firebase authentication and Cloudinary service. And a skateboard builder that allows users to build their custom complete skateboard.

Project Highlight


Storefont Overview:
The navigation emphasizes different product categories, so customers can easily find the products. There are also brand pages that customers can access when they click on the brand logo on the product page. Pagination is added to all product collection pages. Also, there is the account info page, so customers can login to see their order history.



Custom Skateboard builder:
Customers can select items for each part of the skateboard to build a custom complete. It hopes to bring the offline skate shop experience to the customer. After completing the selection, the customer can see the rendered picture to get a sense of what to expect. The image rendering is implemented by using Cloudinary Transformation API. If the customer is happy with the result, he or she can seamlessly add all items to the cart by one click.



Product Page:
Customer can use the picker to select different options, and the price and quantity of the selected variant will change accordingly. Stock quantity is validated before adding to cart, if customer tries to add more quantity than the available stock, a pop-up alert will be shown to indicate the current stock level (the pop up cannot be captured by screen capture) If the customer chooses an out-of-stock variant, the add to cart button will be disabled and show out of stock instead.



Checkout:
When customers check out the cart items, they need to login or register a customer account. After logged in, the customers can confirm the order summary and proceed to payment. The payment service is integrated with Stripe, customers will be directed to the prebuilt, Stripe checkout page. After customers completed the payment, they will be redirected to store and view the order status.



Admin Panel Overview:
The dashboard in the admin panel gives highlights on today's sales and compares them to yesterday's. It also includes three charts showing hourly sales of today, daily sales of this month, and monthly sales of this year, so the admin can monitor the trends over time. The admin can view customer info, edit orders, also add and edit brands, and products.



Create Product:
The creation form features a drag-and-drop image uploader and the admin can select existing brands and categories from the dropdown. After the admin input the options and the respective value for the product, input fields are rendered for the admin to input variant information such as price and stock.



Edit Product:
All fields are editable, the UI is same as the create product page.
